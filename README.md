# Airports Exploratory Data Analysis (EDA)

This project is part of the CodeAlpha Data Analytics Internship.

## Task
Perform Exploratory Data Analysis (EDA) on a global airports dataset.

## Files Included
- airports.csv (original dataset)
- airports_cleaned.csv (cleaned dataset)
- eda_airports.py (Python EDA script)
- top_countries.png (Visualization)
- top_timezones.png (Visualization)
- Airports_EDA_Report.docx (Final EDA Report)

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code

## Key Insights
- A few countries dominate global airport counts
- Airports are concentrated in a few major time zones
- Some data quality issues were identified and cleaned
